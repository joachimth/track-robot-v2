📝 Manual Flash (esptool)

Download binaries from GitHub Releases, then: 

esptool.py --chip esp32 --port /dev/ttyUSB0 --baud 460800 \
  --before default_reset --after hard_reset write_flash \
  0x1000  bootloader.bin \
  0x8000  partition-table.bin \
  0x10000 track-robot.bin

🎮 After Flashing

  1. Open a serial monitor at 115200 baud and verify boot messages
  2. Hold PS + Share on the PS4 controller to enter pairing mode
  3. The ESP32 scans ~10 s — it connects automatically when the controller is found
  4. Press Options on the controller to arm the motors
  5. Drive with the left stick — Cross for emergency stop
