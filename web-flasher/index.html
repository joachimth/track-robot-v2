📝 Manual Flash (esptool)

Download binaries from GitHub Releases, then: 

esptool.py --chip esp32 --port /dev/ttyUSB0 --baud 460800 \
  --before default_reset --after hard_reset write_flash \
  0x1000  bootloader.bin \
  0x8000  partition-table.bin \
  0x10000 track-robot.bin

⚙️ Configure After Flashing

The robot starts an access point you can connect to immediately after flashing.

  1. Connect your phone/PC to the WiFi network TrackRobot-Setup (password: trackrobot)
  2. Open http://192.168.4.1/ in your browser
  3. Use the WiFi tab to connect the robot to your home network (optional)
  4. Use the Config tab to tune drive parameters (deadzone, max speed, slow-mode factor)
  5. Use the Control tab to ARM and manually drive the robot via HTTP

The robot always keeps the TrackRobot-Setup AP active as a fallback, so you can reach the web UI even if home WiFi is unavailable.

🎮 PS4 Controller — Quick Start

  1. Hold PS + Share on the PS4 controller to enter pairing mode (light bar flashes rapidly)
  2. The ESP32 auto-connects — look for "Controller connected" on the serial monitor
  3. Press Options (≡) to arm the motors
  4. Drive with the left stick — Cross (×) for emergency stop
  5. Hold L1 for slow mode (50% speed)
